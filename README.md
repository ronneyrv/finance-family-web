# Finance Family Web

Frontend web da aplicação Finance Family, uma plataforma de gestão financeira familiar integrada à Finance Family API.

A aplicação é otimizada para uso em desktop e oferece uma experiência totalmente responsiva para dispositivos móveis.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint
- Prettier

## Requisitos

Para executar o projeto localmente, é necessário ter instalado:

- Node.js 22+
- npm 10+

## Configuração do ambiente

Instale as dependências:

```bash
npm install
```
